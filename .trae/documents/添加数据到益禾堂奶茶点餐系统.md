# 添加数据到益禾堂奶茶点餐系统

## 计划步骤

### 1. 数据库初始化检查
- 检查数据库是否已经创建
- 如果未创建，执行数据库初始化脚本
- 验证初始数据是否已导入（管理员账号、默认分类等）

### 2. 添加基础数据
- **添加商品分类**：使用 `/api/admin/category/save` 接口
- **添加商品**：使用 `/api/product/admin/add` 接口，包含商品基本信息和规格
- **添加优惠券**：使用 `/api/admin/coupon/save` 接口
- **添加轮播图**：使用轮播图相关接口
- **添加促销活动**：使用促销活动相关接口

### 3. 测试数据添加
- 验证添加的数据是否正确保存到数据库
- 测试查询接口是否能正确返回添加的数据
- 确保数据之间的关联关系正确

### 4. 数据管理建议
- 提供数据添加的最佳实践
- 说明数据之间的关联关系
- 提供常见问题的解决方案

## 预期结果
- 系统中包含完整的商品数据
- 数据库中包含必要的基础数据
- 所有数据添加接口正常工作
- 用户可以通过系统管理后台查看和管理添加的数据
